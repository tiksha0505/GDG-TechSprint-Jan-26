# CampusCerts | Digital Certificate System

CampusCerts is a centralized, secure platform for managing student certificate requests digitally. It eliminates paperwork, speeds up approval, and ensures instant delivery of certificates. The system is designed for both students and educational institutions, providing transparency, real-time tracking, and automated notifications.

1. Features

- Digital Certificate Requests: Students can submit certificate requests online.

- Admin Validation: Requests are verified and approved by institutions.

- Certificate Delivery: Certificates are sent securely via email.

- Metrics Dashboard: Displays total requests, certificates sent, pending requests, and average turnaround time.

- Responsive Design: Works seamlessly on desktop and mobile devices.

- Security & Verification: Ensures authenticity of certificates and safe handling of student data.

2. Tech Stack

- Frontend: HTML, CSS, JavaScript (vanilla)

- Backend: Google Apps Script (for form submission & metrics)

- Data Storage: Google Sheets

- Deployment: Hosted via GitHub Pages (frontend) and Google Apps Script (backend)

3. Usage

- Students fill out the digital form with their details.

- Requests are auto-validated and approved by admins.

- Approved certificates are sent via email.

- Metrics update in real-time on the dashboard.
